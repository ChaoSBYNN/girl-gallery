1.项目构架
	SailsJS + Jquery + Ejs

2.网站介绍
    |-- 首页会随机生成32幅美女图，点击会进入详情页面
    
3.建站原因
    |-- 一次意外看到一款非常好的jquery图片插件，于是就想利用这块插件做一个图片展示的小作品
        但是图片的来源成了大问题，于是在百度上搜索关键词'高清图片',之后就发现了'http://www.22mm.cc/'
        这个网站，网站的图片暴露尺度不大，大多是清纯mm。于是乎我就'盯上了'这个网站，将它的图片爬下来。
        当然，并没有去将爬下来的图片下载到本地，只是找到图片的src即可。
        
4.备注
    |-- 由于网站的图片都是由其他站点提供，所以我并不能保证当你打开网站的首页时，你能够看到图片。

	|-- 感谢'Bootstrap-Image-Gallery'这款插件，给了我很大的启发。
    |-- 今天晚上南京青奥会开幕，虽然就在家门口举办，但是很遗憾，没能去成，不过，通过5个小时的奋战，

        umei-details.js，如果你也想轻松看美女，可以参考参考。
